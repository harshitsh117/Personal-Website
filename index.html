Face Mash

A long game to choose which one we like.

This is a game inspired by "Who would you rather", a game segment played on The Ellen Degenerous show. In this game we get two pictures among which we have to choose the one we like. The choosen one remains there but the other one is replaced with new one. This game goes on until there are no photos left.
